# デザイン方針 — 06 Marginalia

サイト全体の世界観の方針をまとめる。検討用モックは `design/`（`pnpm design` で閲覧）。
採用案は **06 Marginalia**（`design/06-marginalia.html`）。本ドキュメントはその仕様で、
本実装（`src/styles.css` ほか）はここに従う。

## コンセプト

「落ち着いてるけど遊び心」。元の neutral grey + 琥珀（tsuka-ryu.dev / fumadocs neutral
プリセット由来）はテンプレ感が強く、本人の声が出ていなかった。検討の結果、
01 Glitch Stack（叫ぶグリッチ）と 05 Playground（丸ゴシックの遊具）の中間として、
**乾いたユーモア＋アフォリズム＋飾らなさ**を下敷きにした静かな遊びに着地した。

遊び心は「叫ばない」方向へ。大きな衝撃演出はやめ、手描きアスタリスク（✳）・控えめな
ホバー・余白の書き込み（marginalia）で効かせる。

## カラートークン

| トークン   | ライト    | ダーク    | 役割                                                             |
| ---------- | --------- | --------- | ---------------------------------------------------------------- |
| `--paper`  | `#FAF6EC` | `#14110C` | 背景（温かい紙）                                                 |
| `--ink`    | `#1C1813` | `#EDE6D8` | 本文テキスト                                                     |
| `--soft`   | `#837A6C` | `#968C7B` | 補助テキスト（日付・説明・タグ）                                 |
| `--line`   | `#E7DECF` | `#2A2419` | 罫・境界・背景ドット                                             |
| `--accent` | `#C2540C` | `#E8833C` | くすませた琥珀（01 のヒート由来だが落ち着かせる）。ホバー/マーク |
| `--ink2`   | `#46557A` | `#8FA0C8` | 万年筆インク色。リンク前置き `#`・献辞・グリッチの片側           |

ライト/ダークは `prefers-color-scheme` で切り替える。

## タイポグラフィ

| 役割    | フォント                               | 用途                                                                           |
| ------- | -------------------------------------- | ------------------------------------------------------------------------------ |
| display | **Bricolage Grotesque**（500/600/700） | 見出し・記事タイトル・ブランド。わざと不揃いなグロテスクで「機械的でない」表情 |
| body    | **Zen Kaku Gothic New**（400/500）     | 日本語本文。少し表情のある可読フェイス                                         |
| mono    | **JetBrains Mono**（400/500）          | 日付・タグ・ナビ等のユーティリティ。「ターミナルに住んでいる」気配を構造で示す |

- 本文の基準は **17px（1.0625rem）/ line-height 1.7**。
- 列幅 `--max: 740px`。日本語で `max-width: NNch` は使わない（全角だと 1ch≒2 文字幅で
  極端に窮屈に折り返すため、`rem` で持つ）。

## シグネチャ

- **手描きアスタリスク ✳** — SVG で 3 本の線をわずかに歪ませた自前マーク。
  - ブランド名の後ろ（ホバーでゆっくり回転）。
  - ヒーロー見出しの後ろ。
  - セクション区切り（本の「\* \* \*」シーンブレイク）。記事の間にも `✳` を置く＝
    罫線ではなく余白＋記号で分ける。構造でありウィット。
- **ヒーローのホバーグリッチ** — `tsuka-ryu` は常時は静か。**ホバーした瞬間だけ**短く
  スライス＋ `--accent`×`--ink2` の二色ズレが走る（01 の全画面演出ではなく、触れたときの
  小さな反応に留める）。

## 背景

- 薄い **ドットグリッド**（点描）。`radial-gradient` の点を `1.05rem` 間隔・`--line` 65%
  程度で密に敷く。方眼（格子）は宿題感が出るため不採用。
- 「星の瞬き（点滅）」も試作したが、**チラつきを嫌って不採用**。静的なドットに確定。

## レイアウト

- 記事一覧は **marginalia レイアウト**: 日付を本文の左マージンに逃がし、本文（タイトル・
  説明・タグ）は 1 列に集約する。
  - 日付列は `max-content`（日付幅ぶんだけ）・左揃え・`white-space: nowrap`、本文との
    間隔は `1.25rem`。日付の表記は `2026.06.07`（ドット区切り・スペースなし。中点
    `·` は間延びするため不採用）。
  - 狭い画面（`max-width: 640px`）では日付を本文の上に戻す。
- タグは `#tag` の等幅チップ（`#` を `--ink2`）。

## コピー

- **ヒーロー** = `tsuka-ryu✳` ＋ タグライン「キーボードを叩く、何かが壊れる。」のみ。
  長い説明・アイブロウは置かない（自虐コピーは主役の H1 にすると寒いが、名前の下の
  タグラインなら効く、という整理）。
- **フッター** = `——そういうものだ。`（カート・ヴォネガット『スローターハウス5』邦訳の
  合いの手。2 倍ダッシュ `——` で献辞風に）。「何かが壊れる。」→「そういうものだ。」が
  一枚の中で静かに呼応する。英語 `so it goes.` は符牒だと伝わりにくいため邦訳を採用。

## アクセシビリティ

- `prefers-color-scheme` でライト/ダーク対応。
- `prefers-reduced-motion: reduce` で、ブランドマークの回転・ヒーローのホバーグリッチを
  無効化する（点滅系は元々不採用）。
- キーボードフォーカスの可視化は本実装時に担保する。

## 検討した代替案（`design/`）

| #   | 案           | 不採用の理由                                                     |
| --- | ------------ | ---------------------------------------------------------------- |
| 01  | Glitch Stack | グリッチが「叫びすぎ」。方向性の出発点として採用要素は 06 に吸収 |
| 02  | Minimal      | 静かすぎて遊び心が出ない                                         |
| 03  | Terminal     | ギークだが長文記事の全面モノスペースが読みづらい                 |
| 04  | Riso Poster  | 版ズレ＝壊れの翻訳は秀逸だが個人ブログには強すぎ                 |
| 05  | Playground   | 丸ゴシックがキュート寄り。落ち着きが足りない                     |
